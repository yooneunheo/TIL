### 06. 두 수의 곱

<br>

##### 두 실수 A, B를 입력받아 그 곱을 출력하는 프로그램을 작성하세요.

<br>

- 입력 설명 :
  - 첫번째 줄은 두 수 A, B가 주어진다.
  - 두 수는 -100부터 100이하의 실수이다.
  - 두 수는 소수점 둘 째자리까지만 입력된다.
    <br>
- 출력 설명:
  - 첫번째 줄에 곱을 출력한다
  - 소숫점 셋째자리에서 반올림한다.

---

<br>

```c
#include <stdio.h>

int main() {

	double a, b, c;
	scanf("%lf %lf", &a, &b);
	c = a * b; // 순서대로 작성한다.
	printf ("%0.2lf", c); // 소수 둘째자리까지 출력
	return 0;
}
```

- double 이면 제어문자는 lf 이다.
