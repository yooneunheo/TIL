### 10. 구구단 출력하기

<br>

##### 자연수 N이 입력되면 구구단 중 N단을 출력하는 프로그램을 작성하세요.

<br>

- 입력 설명 :
  - 첫 번째 줄에 100이하의 자연수 N이 입력된다.
    <br>
- 출력 설명:
  - 구구단을 출력한다.

---

<br>

<나의 풀이>

```c
#include <stdio.h>

int main() {

	int i, n;
	scanf("%d", &n);

	for(i=1; i<10; i++) {
		printf("%d * %d = %d\n", n, i, n*i);
	}
	return 0;
}
```

<br>

<강사의 풀이>

```c
#include <stdio.h>

int main() {

	int i, n;
	scanf("%d", &n);

	for(i=1; i<=9; i++) {
		printf("%d * %d = %d\n", n, i, n*i);
	}

```
