### 05. 두 수의 합

<br>

##### 두 자연수 A, B를 입력받아 더한 결과를 출력하는 프로그램을 작성하세요.

<br>

- 입력 설명 :
  - 첫번째 줄은 두 자연수 A, B가 주어진다.
  - 두 정수는 1<= A,B <= 100 이다.
    <br>
- 출력 설명:
  - 첫번째 줄에 두 수를 더한 결과를 출력한다.

---

<br>

```c
#include <stdio.h>

int main() {

	int a, b, c;
	scanf("%d %d", &a, &b);
	c = a + b; // 순서대로 작성한다.
	printf ("%d + %d = %d", a, b, c);
	return 0;
}
```

- int a, b, c; 다음에 바로 c = a + b;를 입력하면 인식이 안 됨.
- 반드시 계산 순서대로 작성해야한다.
