### 03. 2차원 배열 출력2

<br>

##### 10 이하의 자연수 N이 입력되면 N\*N 크기의 2차원 배열을 이용하여 아래와 같이 출력하는 프로그램을 작성하세요.

<br>

- 입력 설명 :
  - 첫 번째 줄에 10이하의 자연수 N이 주어진다.
    <br>
- 출력 설명:
  - 2차원 배열을 출력한다.

---

<br>

```c
#include <stdio.h>
int a[11][11]; // 전역변수 선언 시 0으로 초기화
int main() {
int n, i, j, k=1;
scanf("%d", &n);

for(i=0; i<n; i++) {
    for(j=i; j>=0; j--) {
        a[i][j] = k;
        k++;
    }
}

for(i=0; i<n; i++) {
	for(j=0; j<n; j++) {
		printf("%3d", a[i][j]);
	}
	printf("\n");
}

return 0;
}
```
